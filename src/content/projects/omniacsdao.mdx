Omni Analytics Group is where I do consulting work — primarily R, Shiny apps, statistical reporting,
and custom training. We've built interactive tools like the
[bullet-analyzer Shiny app](https://labs.omnianalytics.org/bullet-analyzer/) and teach a course
called *Creating and Designing Interactive Web Applications with Shiny*.

OmniacsDAO is the open-source arm: R and Python packages for Numerai tournament modelling,
Uniswap analytics, Aave growth dashboards, letter-sequence generators, and experiment orchestration.
If you find `Rnumerai` or `uniswappeR` useful, that's us.
