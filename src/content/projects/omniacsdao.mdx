Omni Analytics Group was where I cut my teeth as a professional data scientist — primarily R,
Shiny apps, statistical reporting, and custom training. From 2017 to 2021 I led the data-science
practice as Chief Data Scientist. We built interactive tools like the
[bullet-analyzer Shiny app](https://labs.omnianalytics.org/bullet-analyzer/) and delivered a course
called *Creating and Designing Interactive Web Applications with Shiny*.

OmniacsDAO grew out of that work as the open-source arm: R and Python packages for Numerai
tournament modelling, Uniswap analytics, Aave growth dashboards, letter-sequence generators, and
experiment orchestration. It's community-maintained now — if you find `Rnumerai` or `uniswappeR`
useful, that lineage starts here.
